Requisitos de conocimiento para manejar el sistema

Antes de utilizar el sistema R.A.S.C., se recomienda tener conocimientos básicos en computación y navegación web. Además, es necesario contar con el siguiente equipo y software:

Hardware: Procesador de al menos 1.5 GHz, 2.5 GB de RAM, y 500 MB de espacio en disco duro.

Software: Navegador web compatible (Google Chrome, Mozilla Firefox).

Requerimientos de hardware y software

REQUERIMIENTOS MÍNIMOS DE HARDWARE

Procesador: Procesador de al menos 1.5 GHz, preferiblemente de dos núcleos.

Memoria RAM: 2.5 GB de RAM.

Espacio en Disco Duro: 500 MB de espacio libre en el disco duro para el almacenamiento de datos temporales y caché del navegador.

Conexión a Internet: Conexión estable a Internet con una velocidad de al menos 25 Mb para una navegación fluida.

REQUERIMIENTOS MÍNIMOS DE SOFTWARE

Privilegios de administrador.

Sistema Operativo: Windows 8.1 o superior. MacOS 10.14 (Mojave) o superior. Android 10 o superior.

Navegador Web: Google Chrome versión 90 o superior. Mozilla Firefox versión 88 o superior. Microsoft Edge versión 90 o superior.

Conexión a Internet: Navegador compatible con HTML5 y JavaScript habilitado. Conexión estable a Internet para el acceso a las funcionalidades del aplicativo y actualización de datos en tiempo real.

Glosario

R.A.S.C.: Acrónimo de la aplicación web/móvil diseñado para registrar y notificar avistamientos de especies peligrosas en el Centro de Biotecnológico del Caribe (CBC). Su objetivo es mejorar la seguridad y promover la conservación de la fauna local.

Aplicación web/móvil: Programa informático accesible a través de internet (web) o dispositivos móviles (smartphones, tablets) que facilita ciertas funciones o servicios.

Centro de Biotecnológico del Caribe (CBC): Institución o área de investigación dedicada a la biotecnología ubicada en el Caribe, donde se centra el uso de la aplicación R.A.S.C.

Avistamientos de especies peligrosas: Observación de animales o plantas que representan un riesgo para la seguridad o la salud de las personas o el medio ambiente.

Seguridad de las personas: Protección de la integridad física y emocional de los individuos frente a peligros potenciales, en este caso, relacionados con especies peligrosas.

Conservación de la fauna local: Protección y preservación de las especies animales que habitan en una región específica, en este caso, el área del CBC.

Respuesta rápida y eficaz: Acción inmediata y efectiva para manejar situaciones de emergencia, minimizando el riesgo y los daños.

Gestión de incidentes: Proceso de identificación, evaluación y manejo de situaciones problemáticas, en este caso, relacionadas con avistamientos de especies peligrosas.

Problema de verificación de avistamientos sin imagen: Dificultad para confirmar la presencia de una especie peligrosa si no se cuenta con una evidencia visual o fotográfica del avistamiento.

Módulos de la aplicación: Componentes o secciones específicas de la aplicación R.A.S.C. que cumplen diferentes funciones, como la gestión de incidentes.

Evaluación y manejo de situaciones peligrosas: Proceso de analizar y controlar incidentes que involucran especies potencialmente peligrosas para minimizar riesgos y asegurar una respuesta adecuada.

Proyecto: El desarrollo e implementación de la aplicación R.A.S.C. para registrar y notificar avistamientos de especies peligrosas en el Centro de Biotecnológico del Caribe (CBC), con un objetivo específico y una duración definida.

Requerimiento: Necesidades documentadas sobre las funcionalidades de la aplicación R.A.S.C., como la capacidad de gestionar avistamientos y verificar su veracidad, esenciales para cumplir con los objetivos de seguridad y conservación.

Sistema: La infraestructura tecnológica y organizativa que soporta la aplicación R.A.S.C., incluyendo la plataforma web/móvil y los módulos necesarios para la gestión de incidentes y emergencias biológicas.

Software: es el conjunto de programas y aplicaciones que permiten a un dispositivo realizar tareas específicas. Incluye sistemas operativos, aplicaciones (como navegadores y procesadores de texto), herramientas de desarrollo, utilidades y sistemas de bases de datos.

Hardware: son los componentes físicos de una computadora, como el procesador, la memoria, el disco duro, la pantalla y el teclado.
